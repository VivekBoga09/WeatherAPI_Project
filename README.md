# 🌦️ Weather Forecast App

A responsive **Weather Forecast Web Application** built with **React.js** that displays real-time weather information and a 5-day forecast for any city using the OpenWeather API.

---

## 🚀 Features

- 🔍 Search weather by city name
- 🌡️ Display current temperature and weather conditions
- 📅 5-day weather forecast
- 💨 Shows humidity, wind speed, and "feels like" temperature
- 🎨 Modern **glassmorphism UI with animations**
- 📱 Fully responsive design
- ☁️ Weather icons from OpenWeather
- 🌐 Deployed online using Netlify

---

## 🛠️ Technologies Used

- **React.js**
- **JavaScript (ES6)**
- **CSS3 (Glassmorphism + Animations)**
- **OpenWeather API**
- **React Hook Form**
- **Netlify (Deployment)**
